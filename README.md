# FootFlex - Foot Exercise App

A beautiful, modern web application designed to help people with foot problems strengthen and rehabilitate their feet through guided exercises.

## Features

- **12 Expert-Designed Exercises** - Curated exercises targeting different foot problems
- **4 Exercise Categories** - Stretching, Strengthening, Mobility, and Relaxation
- **Guided Workouts** - Pre-built workout routines (Quick Relief, Morning Stretch, Evening Relax)
- **Workout Timer** - Built-in timer for each exercise with play/pause controls
- **Progress Tracking** - Track your workouts, exercise count, and time spent
- **Beautiful Modern UI** - Clean, accessible design with smooth animations
- **Fully Responsive** - Works on desktop, tablet, and mobile devices
- **Local Storage** - Your progress is saved locally in your browser

## Getting Started

Since this is a vanilla HTML/CSS/JavaScript application, no installation is required. Simply:

1. Open `index.html` in your web browser
2. Start exploring exercises or begin a workout

## Exercise Categories

### Stretching
- Towel Stretch

### Strengthening
- Toe Curls
- Arch Lifts
- Marble Pickups
- Heel Raises

### Mobility
- Ankle Circles
- Toe Spreads
- Big Toe Stretch

### Relaxation
- Foot Roll
- Foot Massage

## Workout Options

- **Quick Relief** - 5 minutes, 3 exercises
- **Morning Stretch** - 10 minutes, 5 exercises
- **Evening Relax** - 15 minutes, 6 exercises
- **Custom Workout** - Create your own routine

## Technical Details

- **No Build Process Required** - Pure HTML, CSS, and JavaScript
- **Icons** - Lucide icons (loaded via CDN)
- **Fonts** - Inter font family (loaded via Google Fonts)
- **Storage** - Progress saved to localStorage
- **Browser Support** - Modern browsers (Chrome, Firefox, Safari, Edge)
